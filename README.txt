Simple Anchor mod [simple_anchor]


A node that adds itself to the forceload list on placement, keeping the map block it is in loaded when players are not near by. It removes itself from the list on destruction. Punch it to see the area that is affected.

Area display based on Protector Redo by TenPlus1
